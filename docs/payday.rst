.. _payday:

======
PayDay
======

This is the cog guide for the payday cog. You will
find detailed docs about usage and commands.

``[p]`` is considered as your prefix.

.. note:: To use this cog, load it by typing this::

        [p]load payday

.. _payday-usage:

-----
Usage
-----

Customizable PayDay system
